# EduPrompt Studio

Aplicación web estática (HTML/CSS/JavaScript) para docentes universitarios que desean crear, evaluar y entrenar prompts profesionales.

## Funcionalidades

- Prompt Builder guiado con campos clave y acciones (copiar, limpiar, guardar historial, exportar TXT).
- Biblioteca de plantillas docentes por categoría.
- Evaluador de prompts con rúbrica (0-100), recomendaciones y versión mejorada.
- Simulador de entrenamiento con retos prácticos.
- Historial con filtros, búsqueda, exportación, copia y borrado.

## Estructura

- `index.html`: dashboard principal.
- `styles.css`: estilo visual tipo AI Studio.
- `script.js`: router SPA y estado global.
- `pages/`: vistas principales.
- `components/`: utilidades UI reutilizables.
- `services/`: lógica de negocio y localStorage.

## Uso

Abre `index.html` en el navegador o publícalo directamente en GitHub Pages.
